Current Evidence Basis

Our current demonstration uses NASA's publicly available C‑MAPSS turbofan engine dataset—a canonical benchmark for degradation trajectory analysis. This dataset helps us test multi‑signal correlation logic and lead‑time estimation; however, it is not a substitute for field validation on water or building infrastructure.

Dataset

NASA C‑MAPSS

Run‑to‑failure simulations of turbofan engines used to test prognostics models. Suitable for evaluating multi‑signal deviation detection and algorithm development.

Scope

Analytical Prototype

Demonstrates telemetry ingestion, sustained drift detection, and baseline comparisons. Evidence basis is a proof of direction, not a production validation.

Limitations

Field Validation

Current evidence does not reflect live water or building systems. The next step is to validate on domain‑specific telemetry from municipal, commercial, and industrial sites.

What Neraium Provides

Neraium delivers systemic infrastructure intelligence for investors and operators seeking long‑term resilience and transparency. Our read‑only platform ingests existing signals, correlates multi-domain deviations, and generates evidence‑grade records for evaluation and investment analysis.

CAP-01

Read‑Only Ingestion

Connect to existing telemetry streams—no new sensors or control links required. All data remains within the protected perimeter.

CAP-02

Multi‑Signal Analysis

CAP-03

System Model

Neraium is designed as a three‑layer pipeline, keeping ingestion, analysis, and review strictly separate for security and accountability. This model ensures clarity for investors and operators alike.

Layer 01
Ingest
Normalize and time‑align your existing telemetry, preserving raw data lineage for auditors and investors.
→
Layer 02
Analyze
Run multi‑signal logic to detect persistent deviation, baseline drift, and composite scoring across all monitored domains.
→
Layer 03
Review
Deliver structured insights and permanent records for investors and operators to review. No autonomous control—human authority remains intact.
