Core Methods

How the platform models behavior

Neraium does not depend on isolated thresholding or purely black-box forecasting. It models system structure directly through relationships between variables.

Mahalanobis Distance

Measures deviation in multivariate system space rather than evaluating each variable independently.

Covariance Modeling

Captures relationships between system variables so structural drift can be detected across correlated signals.

Adaptive Baselines

Expected system structure evolves with valid behavior, preserving operational relevance over time.

Interpretation

Constraining statistical output

Statistical output alone is not operational intelligence. Interpretations must satisfy physical plausibility, system consistency, and operator review.

Physical Validity

Signals must remain physically plausible within the infrastructure environment.

Multi-Sensor Agreement

Correlated telemetry must support the interpretation before it is admitted into the intelligence layer.

Operator Review

Surfaced interpretations support operators rather than replacing them. Human judgment remains final.
