The Data Diode Standard.

Neraium utilizes a unidirectional ingestion model. Data can move from infrastructure to the intelligence layer, but no physical or logical path exists to issue commands back to the source.

Ingestion Protocols

  • [+] BACnet/SC (Secure Connect)
  • [+] Modbus TCP Polling
  • [+] OPC UA Unidirectional
  • [+] MQTT Sparkplug B

Security Constraints

  • [+] AES-256-GCM Encryption
  • [+] TLS 1.3 Termination
  • [+] Hardware Data Diode
  • [+] Zero-Trust Edge Node
